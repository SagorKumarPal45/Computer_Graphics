1.In the root folder there is zip file mingw64.zip
2.Copy it and paste in the Local disk (c) after that extract it.
3.Open the folder and copy the bin folder path and add it to environment variables in your computer.
4.Now open the full folder in the vs code.
5.Now open the terminal if the terminal is in the PS(Powershell) change it to CMD (Command Promt)
6.Now copy paste this promt To Compile the code :
  "g++ Untitled2.cpp -o app.exe -lbgi -lgdi32 -lcomdlg32 -luuid -loleaut32 -lole32"
7.Happy Codeing :)
